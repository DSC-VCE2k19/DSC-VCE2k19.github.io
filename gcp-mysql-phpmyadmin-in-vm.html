GCP – Native MySQL and PhpMyAdmin in VM

  1. Create Ubuntu VM and go to SSH
  2. Update and install Apache2, php
    1. sudo apt-get update
    2. sudo apt-get install apache2 php libapache2-mod-php
  3. Run Apache Server
    1. sudo systemctl start apache2.service
  4. Create a file and ping that on server
    1. sudo sh -c ‘echo “Welcome” > /var/www/html/phpinfo.php’
  5. Install MySQL
    1. sudo apt-get install -y mysql-server
  6. Improve MySQL installation security
    1. sudo mysql_secure_installation
  7. Connect to MySQL
    1. sudo mysql -u root -p
  8. Install phpMyAdmin
    1. sudo apt-get install phpmyadmin
  9. Configure phpMyAdmin (in GUI kind of thing)
    1. Select apache2
    2. Select yes to use dbconfig common for database setup
    3. Enter the database administrator’s password that you chose during MySQL configuration
    4. Enter a password for phpMyAdmin application
  10. In case of wrong config, reconfigure using
    1. sudo dpkg-reconfigure phpmyadmin
  11. Login into phpMyAdmin at http://EXTERNAL_IP/phpmyadmin using phpMyAdmin as user
  12. Upload file using google console or SSH
  13. Copy/move those files to /var/www/html
  14. Ping http://PUBLIC_IP/index.php
